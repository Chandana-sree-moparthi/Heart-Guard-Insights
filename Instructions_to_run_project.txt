# Project Instructions

## Method #1: Run Project on Your Machine

1. Locate the file **"packages_used_in_project_development.txt"**. This file contains the list of required modules along with their installation commands.
2. Install all required modules by running the following command in your terminal:
   `pip install -r packages_used_in_project_development.txt`
3. Once installation is complete, run the project by executing:
   `python app.py`

---

## Method #2: Run Project Using requirements.txt

1. Install Python on your machine.
2. Open your terminal and navigate to the directory where the project files are stored.
3. Install all required libraries using:
   `pip install -r /path/to/requirements.txt`
4. Once installation is done, run the project with:
   `python app.py`  
   This will start the Flask server. Visit the provided link to enjoy the project!

---

## Method #3: Run Project Using Anaconda Environment

1. Install Anaconda on your machine.
2. Navigate to the directory where the project files are stored and open the Anaconda prompt there.
3. Create an environment using the provided `conda_requirements_yml.yml` file:
   `conda env create -n my_project_env --file conda_requirements_yml.yml`
4. Activate the created environment (`my_project_env`) and run the following:
   `conda activate my_project_env`  
   `python app.py`
